Ridge 回归（L2 正则化）

什么是 Ridge 回归？

Ridge 回归（岭回归） 是线性回归的L2 正则化版本。

核心直觉：

  • 在损失函数中添加权重平方和作为惩罚项
  • 约束权重的大小，防止过拟合
  • 当特征高度相关时特别有用

与普通线性回归对比：

  • 普通回归：最小化 $\sum(y_i - \hat{y}_i)^2$
  • Ridge 回归：最小化 $\sum(y_i - \hat{y}_i)^2 + \alpha \sum w_j^2$

数学原理

1. 普通线性回归：

$$\min_w \sum_{i=1}^n (y_i - w^T x_i)^2$$

2. Ridge 回归（添加 L2 惩罚）：

$$\min_w \sum_{i=1}^n (y_i - w^T x_i)^2 + \alpha \sum_{j=1}^p w_j^2$$

3. 闭式解：

$$w = (X^T X + \alpha I)^{-1} X^T y$$

其中 $I$ 是单位矩阵，$\alpha$ 是正则化强度。

4. 权重收缩效果：

  • 当 $\alpha \to 0$：Ridge 回归 ≈ 普通线性回归
  • 当 $\alpha \to \infty$：所有权重趋近于 0
  • 适度的 $\alpha$ 可以防止过拟合

5. Ridge vs Lasso：

  • Ridge (L2)：权重趋近于 0 但不等于 0（收缩）
  • Lasso (L1)：权重可能精确等于 0（稀疏性）
