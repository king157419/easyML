PCA 主成分分析可视化

什么是 PCA？

主成分分析（Principal Component Analysis, PCA） 是一种降维技术。

核心直觉：

  • 高维数据难以可视化和处理（"维数灾难"）
  • PCA 找到数据方差最大的方向（主成分）
  • 丢弃其他方向，只保留最重要的几个成分
  • 实现降维，减少计算量，同时保留大部分信息

常见应用：图像压缩、特征提取、数据可视化、噪声过滤等。

数学原理

PCA 的核心是计算协方差矩阵，找到特征向量和特征值。

步骤：

  1. 数据中心化（减去均值）
  2. 计算协方差矩阵
  3. 计算特征值和特征向量
  4. 选择 top-k 个主成分
  5. 数据投影到新空间

协方差矩阵：

$$Cov[i,j] = E[(X_i - \mu_i)(X_j - \mu_j)^T]$$

其中：

  • $Cov[i,j]$ 是特征 $i$ 和 $j$ 的协方差
  • $E$ 是特征值组成的对角矩阵
  • $X_i$ 是第 $i$ 个数据点
  • $\mu_i$ 是第 $i$ 个特征的均值
