DBSCAN 密度聚类可视化

什么是 DBSCAN？

DBSCAN（Density-Based Spatial Clustering of Applications with Noise） 是一种基于密度的聚类算法。

核心直觉：

  • 聚类是密度相连的点的集合
  • 可以发现任意形状的聚类
  • 自动识别噪声点
  • 不需要预先指定聚类数量

与 K-Means 对比：

  • K-Means：基于距离，假设聚类是球形的
  • DBSCAN：基于密度，可以发现任意形状

数学原理

1. 核心定义：

  • ε-邻域：点 p 的 ε 邻域是所有与 p 距离 ≤ ε 的点集
  • 核心点：ε-邻域中至少有 MinPts 个点
  • 边界点：ε-邻域中点数 < MinPts，但在某个核心点的邻域内
  • 噪声点：既不是核心点也不是边界点

2. 密度可达性：

  • 点 q 从点 p 密度可达：存在一条链 p=p₁, p₂, ..., pₙ=q，每个 pᵢ 都是核心点，且 pᵢ₊₁ 在 pᵢ 的 ε-邻域内

3. 密度相连：

  • 点 p 和 q 密度相连：存在一个点 o，使得 p 和 q 都从 o 密度可达

4. 聚类定义：

  • 一个聚类是基于密度可达性的最大密度相连点集
  • 聚类中的任何点都从该聚类中的某个核心点密度可达

5. 算法步骤：

  1. 随机选择一个未访问的点 p
  2. 如果 p 是核心点，创建新聚类，递归添加所有密度可达的点
  3. 如果 p 不是核心点，标记为噪声（稍后可能被重新分类为边界点）
  4. 重复直到所有点都被访问
