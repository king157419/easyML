逻辑回归 (Logistic Regression) 可视化

什么是逻辑回归？

逻辑回归是用于二分类问题的经典算法。尽管名字里有"回归"，但它实际上是一个分类算法。

核心区别：线性回归预测连续值（如房价），逻辑回归预测概率（如是不是垃圾邮件）。

核心直觉：通过 Sigmoid 函数将线性输出压缩到 [0, 1] 区间，表示属于正类的概率。

数学原理

逻辑回归使用 Sigmoid 函数将线性输出映射到概率：

$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

其中 z 是线性组合：

$$z = w_1 x_1 + w_2 x_2 + b$$

损失函数（交叉熵）：

$$L = -\frac{1}{n}\sum_{i=1}^{n}[y_i\log(\hat{y}_i) + (1-y_i)\log(1-\hat{y}_i)]$$

🎯 直觉理解：
• Sigmoid 函数将任意实数映射到 (0, 1)，可以理解为概率
• 当 z = 0 时，σ(z) = 0.5，这是决策边界
• 决策边界是线性的（一次函数），但概率变化是非线性的
• 通过最大化似然估计（或最小化交叉熵）学习参数
