线性回归 (Linear Regression) 可视化

什么是线性回归？

线性回归是最基础的机器学习算法之一。它的目标是找到一条最佳直线（或超平面），用来描述因变量和自变量之间的关系。

例如：根据房屋面积预测房价、根据学习时间预测考试成绩等。

核心直觉：想象你在散点图中画一条线，让所有点到这条线的距离总和最小。

数学原理

线性回归的目标是最小化均方误差 (MSE)：

$$MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$$

其中拟合线方程为：

$$\hat{y} = mx + b$$

最优解（闭式解）：

$$m = \frac{\sum(x_i - \bar{x})(y_i - \bar{y})}{\sum(x_i - \bar{x})^2}, \quad b = \bar{y} - m\bar{x}$$

🎯 直觉理解：
斜率 m 控制线的"倾斜程度"——正数向上，负数向下
截距 b 控制线的"起始位置"——与 y 轴的交点
MSE 是每个点到线的垂直距离的平方平均
