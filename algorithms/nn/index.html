神经网络逐层激活可视化

算法简介

神经网络（Neural Networks） 是受人脑神经元结构启发的机器学习模型。它由多层相互连接的"神经元"组成，每层神经元从前一层接收输入，经过加权求和和非线性激活后，输出到下一层。

核心组件：

  • 权重（Weights）：神经元之间的连接强度，通过训练学习得到
  • 偏置（Bias）：每个神经元的偏移量
  • 激活函数：引入非线性，常用的有 ReLU、Sigmoid、Tanh

为什么需要可视化？

神经网络常被称为"黑盒"，因为难以理解内部发生了什么。通过可视化逐层激活，我们可以：1）观察特征是如何逐步提取和转换的；2）诊断训练问题（如梯度消失、神经元死亡）；3）理解网络学到了什么。

数学原理

前向传播：

对于第 l 层的第 j 个神经元：

zⱼ⁽ˡ⁾ = Σᵢ wⱼᵢ⁽ˡ⁾ · aᵢ⁽ˡ⁻¹⁾ + bⱼ⁽ˡ⁾

aⱼ⁽ˡ⁾ = σ(zⱼ⁽ˡ⁾)

其中：

  • zⱼ⁽ˡ⁾ 是第 j 个神经元的加权输入
  • wⱼᵢ⁽ˡ⁾ 是从上一层的第 i 个神经元到当前层的第 j 个神经元的权重
  • bⱼ⁽ˡ⁾ 是偏置
  • σ(·) 是激活函数

激活函数：

  • ReLU: σ(z) = max(0, z) - 最常用，缓解梯度消失
  • Sigmoid: σ(z) = 1/(1+e⁻ᶻ) - 输出在 (0,1)
  • Tanh: σ(z) = (eᶻ-e⁻ᶻ)/(eᶻ+e⁻ᶻ) - 输出在 (-1,1)

网络架构信息

输入层: 2 个神经元

隐藏层: 2 层，每层 4 个神经元

输出层: 1 个神经元

总参数: 0 个权重 + 0 个偏置
