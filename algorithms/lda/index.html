LDA 线性判别分析

什么是 LDA？

LDA（Linear Discriminant Analysis，线性判别分析） 是一种用于降维和分类的监督学习方法。

核心直觉：

  • 找到一个投影方向，使得同类数据尽可能紧凑
  • 同时使不同类数据尽可能分离
  • 最大化"类间距离/类内距离"比率

与 PCA 对比：

  • PCA：无监督，最大化方差
  • LDA：有监督，最大化类别可分性

数学原理

1. 散度矩阵：

  • 类内散度矩阵（Within-class scatter）： $$S_W = \sum_{c} \sum_{x \in c} (x - \mu_c)(x - \mu_c)^T$$
  • 类间散度矩阵（Between-class scatter）： $$S_B = \sum_{c} n_c (\mu_c - \mu)(\mu_c - \mu)^T$$

2. 目标函数：

最大化瑞利商：

$$J(w) = \frac{w^T S_B w}{w^T S_W w}$$

3. 最优解：

$$w^* = S_W^{-1} (\mu_1 - \mu_2)$$

即：类内散度的逆乘以类均值差。

4. 投影：

新特征 = $w^T x$
