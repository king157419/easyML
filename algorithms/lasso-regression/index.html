Lasso 回归（L1 正则化）

什么是 Lasso 回归？

Lasso 回归（Least Absolute Shrinkage and Selection Operator） 是线性回归的L1 正则化版本。

核心直觉：

  • 在损失函数中添加权重绝对值作为惩罚项
  • 能够产生稀疏解：许多权重精确变为 0
  • 自动进行特征选择

与 Ridge 对比：

  • Ridge (L2)：权重收缩到接近 0
  • Lasso (L1)：权重可以精确等于 0

数学原理

1. Lasso 目标函数：

$$\min_w \sum_{i=1}^n (y_i - w^T x_i)^2 + \alpha \sum_{j=1}^p |w_j|$$

2. 几何解释：

  • L1 约束区域是菱形（有尖角）
  • 等高线更容易在尖角处接触
  • 在坐标轴上接触 = 某些权重为 0

3. 坐标下降算法（Coordinate Descent）：

对于每个特征 j：

$$w_j = S(z_j, \alpha) / (\sum_{i=1}^n x_{ij}^2)$$

其中 $S$ 是软阈值函数：

$$S(z, \alpha) = \text{sign}(z) \cdot \max(|z| - \alpha, 0)$$

4. 稀疏性：

  • 当 $|z_j| \leq \alpha$ 时，$w_j = 0$
  • 这是 Lasso 产生稀疏解的关键
